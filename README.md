# 🏥 Hospital Management System

A modern and responsive **Hospital Management System** built to streamline hospital operations including patient management, doctor scheduling, appointments, billing, and medicine inventory.

This project is designed with a clean UI and scalable backend architecture to simulate a real-world healthcare management platform.

# ✨ Features

## 👨‍⚕️ Patient Management

* Add, update, and delete patient records
* Patient medical history tracking
* Search and filter functionality
* Patient profile management

## 🩺 Doctor Management

* Doctor profiles and specialization
* Availability management
* Appointment scheduling

## 📅 Appointment System

* Book appointments
* Appointment status tracking
* Appointment management dashboard

## 💳 Billing System

* Generate invoices
* Payment tracking
* Billing history

## 💊 Medicine Inventory

* Medicine stock management
* Low stock alerts
* Medicine categories

## 🔐 Authentication & Security

* JWT Authentication
* Secure login/signup
* Protected routes
* Role-based access control

## 📊 Admin Dashboard

* Analytics overview
* Revenue statistics
* Patient and doctor insights
* Interactive charts and reports

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/patelsaba919-design/Hospital-Managment-system.git
```

## 2️⃣ Navigate to Project Folder

```bash
cd Hospital-Managment-system
```

## 3️⃣ Install Dependencies

```bash
npm install
```

## 4️⃣ Setup Environment Variables

Create a `.env` file and add:

```env
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
PORT=5000
```

## 5️⃣ Run the Application

```bash
npm run dev
```
